Train a collaborative filtering matrix decomposition recommender using [SparkML’s Alternating Least Squares (ALS)](https://spark.apache.org/docs/2.1.0/ml-collaborative-filtering.html) to batch-compute query-query similarities. This can be used for [items-for-query recommendations](/fusion/5.9/481/items-for-query-recommendations) as well as [queries-for-query recommendations](/fusion/5.9/479/queries-for-query-recommendations).
