This group of APIs lets you manage authentication and permissions in your Fusion instance. You can use these endpoints to control [session](/fusion/5.9/339/sessions-api) keys, manage native [realms user accounts](/fusion/5.9/337/realms-api), and more.

For more information, [view the API specification.](?open-api=true)
