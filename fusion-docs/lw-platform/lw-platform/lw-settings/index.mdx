The Settings feature of Lucidworks Platform lets you:

* Create and manage user accounts, including setting roles, permissions, and authentication for users in Lucidworks Platform.
* Configure integrations that connect Fusion to Lucidworks Platform and other software to extend and enhance your workflows.

To access the Settings screen, complete the following:

1. Sign in to [Lucidworks Platform](https://platform.lucidworks.com) as a workspace owner.
2. Verify the correct workspace is selected and click the **Settings** icon.
3. Access any of the following from the Settings screen:

   * To create and manage user accounts and user permissions, click **Users & Provisioning > Users**. For more information, see [Users and Provisioning](/lw-platform/platform/00j8nx/users-and-provisioning).
   * To configure single sign-on authentication, click **Users & Provisioning > Authentication**. For more information, see [Single sign-on authentication](/lw-platform/platform/cpq1c3/single-sign-on-authentication).
   * To configure and manage Fusion integrations to Lucidworks Platform, Commerce Studio, and Analytics Studio, click **Fusion Integrations**. For more information, see

     + [Lucidworks Platform integrations with Fusion](/lw-platform/platform/04gi1q/integrations)
     + [Commerce Studio integration with Fusion](/lw-platform/platform/zpqyc5/commerce-studio-integration-with-fusion)
     + [Configuration Promotion tool integration with Fusion](/lw-platform/platform/qjns2v/configuration-promotion-tool-integration-with-fusion)

Integrations between Lucidworks AI and Fusion are configured using the **Products** screen when you select an application and then click **Integrations**. For more information, see [Lucidworks AI integrations](/lw-platform/platform/mdlec0/lucidworks-ai-integrations).
