This article provides a feature comparison of Commerce Studio and Predictive Merchandiser.

Both Predictive Merchandiser and Commerce Studio provide features that help your organization enhance and fine-tune searches and discovery on your website. Insights about customer interactions enable personnel such as merchandisers, marketers, and product teams develop and refine queries and campaigns to optimize customer experiences.

| Feature | Predictive Merchandiser | Commerce Studio | Notes |
| --- | --- | --- | --- |
| Integration with Fusion | ✅ | ✅ | Commerce Studio supports [integrating with Fusion](/lw-platform/platform/zpqyc5/commerce-studio-integration-with-fusion). |
| Modern user interface | ✘ | ✅ | The main dashboard is called [Visual Editor](/lw-platform/cs/l6f36o/visual-editor) in Commerce Studio, and is designed to be intuitive and easy-to-use. |
| Rules management | ✅ | ✅ | In Commerce Studio, rule management can be performed in either the:  * [Ranking Rules UI](/lw-platform/cs/ravw5k/ranking-rules) via a wizard * [Visual Editor UI](/lw-platform/cs/l6f36o/visual-editor) |
| Head/Tail rewrites | ✅ | ✅ | In Commerce Studio, this feature is called [Underperforming Queries](/lw-platform/cs/5zeagj/rewrite#create-a-query-rewrite). |
| Misspelling corrections | ✅ | ✅ | In Commerce Studio, this feature is called [Misspelling Detections](/lw-platform/cs/5zeagj/rewrite#create-a-query-rewrite). |
| Phrase rewrites | ✅ | ✅ | In Commerce Studio, this feature is called [Phrase Detections](/lw-platform/cs/5zeagj/rewrite#create-a-query-rewrite). |
| Synonym additions | ✅ | ✅ | In Commerce Studio, this feature is called [Synonym Detections](/lw-platform/cs/5zeagj/rewrite#create-a-query-rewrite). |
| Remove Words | ✅ | ✅ | In Commerce Studio, this feature is called [Word Removal](/lw-platform/cs/5zeagj/rewrite#create-a-query-rewrite). |
| Custom interface configuration | ✅ | ✅ | Instead of Templates and Zones, Commerce Studio uses [Pages](/lw-platform/cs/t968bf/pages) and Sections that you can set up to contain facets, recommendations, and results. |
| Analytics Studio compatibility | ✘ | ✅ | Predictive Merchandiser uses analytics that are derived from signals collected and indexed in Fusion. Commerce Studio uses [Inline Analytics](/lw-platform/cs/l6f36o/visual-editor#inline-analytics) on catalog item cards in the Visual Editor that are powered by [Analytics Studio](/lw-platform/analytics/10aut8/analytics-studio) and the [signals beacon](/lw-platform/analytics/k649vy/signals-beacon). |
| Simulate customer experience | ✅ | ✅ | Commerce Studio features the [Live Site Simulator](/lw-platform/cs/93f8tp/live-site-simulator), which you can access from the Visual Editor to test rule changes in real time. |
