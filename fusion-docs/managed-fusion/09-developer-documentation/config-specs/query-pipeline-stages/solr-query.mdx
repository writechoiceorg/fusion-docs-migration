The Solr Query stage transforms the Managed Fusion query pipeline [Request](/fusion-pipeline-javadocs/5.9/com/lucidworks/apollo/pipeline/query/request) object into a Solr query and sends it to Solr.

## Configuration

When entering configuration values in the UI, use *unescaped* characters, such as `\t` for the tab character. When entering configuration values in the API, use *escaped* characters, such as `\\t` for the tab character.
</Tip>
