The Parsers API is [deprecated in Managed Fusion 5.9.11](/managed-fusion/5.9/0bs8ji/managed-fusion-5-9-11#deprecations). Use the [Async Parsing API](/managed-fusion/5.9/3uszuk/async-parsing-api) instead.

The Parser API provides CRUD operations for [parsers](/managed-fusion/5.9/0e80mp/parsers).

For more information, [view the API specification.](?open-api=true)

To configure an index pipeline to use a specific parser, see the [Index Pipelines API](/managed-fusion/5.9/m6at2r/index-pipelines-api).

<Note>
In order to see this object within the [Managed Fusion UI](/managed-fusion/5.9/v90jze/object-explorer), it **must** be associated with an app. To do this, create the object using the `/apps` endpoint.
</Note>

For more information about structuring documents for indexing, see [Importing Data with the REST API](/how-to/865/import-data-with-the-rest-api).
