This group of APIs lets you manage authentication and permissions in your Managed Fusion instance. You can use these endpoints to control [session](/managed-fusion/5.9/tu3rmc/sessions-api) keys, manage native [realms user accounts](/managed-fusion/5.9/ivfdzd/realms-api), and more.

For more information, [view the API specification.](?open-api=true)
