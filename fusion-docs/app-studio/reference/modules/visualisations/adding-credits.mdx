To display credits on the chart in the bottom right hand corner, use the Chart Credit tag with the `text` and `href` attributes.

This tag must be nested in the [Chart Display tag](/fusion/5.9/3151/displaying-a-chart).

```xml
<chart:display response="response" title="Current Products" type="column" facet-name="current_products">
	<chart:credit href="http://twigkit.com" text="twigkit.com"></chart:credit>
</chart:display>
```
