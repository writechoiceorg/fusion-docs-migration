Visualize the top filters of a facet as a tag cloud.

<Frame>![tagcloud](/assets/images/appkit/4.2/tagcloud.png)</Frame>

## Usage

To add a tag cloud, specify the response, which facet to display, and (optionally) how many tags to display.

```xml
<chart:tagcloud response="response" facet-name="facet"></chart:tagcloud>
```

To see all related attributes, see the [chart:tagcloud](/fusion/5.9/1071/chart-tagcloud) tag documentation.
