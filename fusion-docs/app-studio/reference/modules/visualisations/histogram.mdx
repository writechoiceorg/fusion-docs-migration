The histogram visualization displays the filters of a facet as a left-to-right series of bars.

## Usage

Simply pass a facet name into the histogram visualization to see that facet presented as a histogram chart.

```xml
<chart:histogram facet-name="facet" show="20"></chart:histogram>
```

To see all related attributes, see the [chart:histogram](/fusion/5.9/1179/chart-histogram) tag documentation.
