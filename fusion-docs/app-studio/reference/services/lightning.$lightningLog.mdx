## Description

Twigkit factory for logging.

## Methods

### debug(file, message)

Method to debug

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| file | string | The file the message is coming from |
| message | string | The message to be displayed |

#### Example

```js
$lightningLog.debug(instance,message);
```
