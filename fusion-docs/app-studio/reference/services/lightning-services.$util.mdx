## Description

Twigkit factory with utility functions for Development.

## Methods

### hasChildren(transclude)

  Method to check if the element has children.

  #### Parameters

  | Param | Type | Details |
  | --- | --- | --- |
  | transclude | Object | The transclude function from the elements post or pre linking function. |

  #### Returns

  |  |  |
  | --- | --- |
  | boolean | True if element has children. |

  #### Example

```js
  $util.hasChildren(transclude);```
