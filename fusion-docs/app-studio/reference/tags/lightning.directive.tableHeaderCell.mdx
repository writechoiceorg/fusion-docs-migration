## Description

Creates a table header cell `<th>`

## Usage

as element:

```xml
<table:header-cell
       [label="{string}"]
       [name="{string}"]
       [sort="{boolean}"]>
</table:header-cell>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| label  *(optional)* | **string** | Label for the header. |
| name  *(optional)* | **string** | Name of the field to sort. |
| sort  *(optional)* | **boolean** | Whether to enable sorting. |
