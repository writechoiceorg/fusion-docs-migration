## Description

The toggle panes directive

## Usage

as element:

```xml
<toggle:panes
       id="{string}"
       [allow-multiple="{boolean}"]>
</toggle:panes>
```

### Directive info

* This directive creates new scope.

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| id | **string** | Id of the element |
| allow-multiple  *(optional)* | **boolean** | Whether or not multiple widgets should be shown at once. Default: false. |
