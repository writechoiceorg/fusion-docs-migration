## Description

Creates a tooltip.

## Usage

as attribute

```xml
<ANY tooltip="{string}">
   ...
</ANY>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| tooltip | **string** | The text to display in the tooltip |
