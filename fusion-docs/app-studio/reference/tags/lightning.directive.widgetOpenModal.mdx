## Description

Applies a class to the target when toggled on.

## Usage

as attribute

```xml
<ANY widget-open-modal="{string}">
   ...
</ANY>
```

### Directive info

* This directive creates new scope.

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| widget-open-modal | **string** | The ID of the modal to open |

## Example

#### Source

```xml
<button class="tk-stl-button" widget-open-modal="id">Open Modal</button>
<widget:modal title="Hello World" id="id">
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. </widget:modal>
```
