## Description

Adds grid classes to element

## Usage

as element:

```xml
<layout:grid
       [fixed="{string}"]>
</layout:grid>
```

as attribute

```xml
<ANY layout-grid
     [fixed="{string}"]>
   ...
</ANY>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| fixed  *(optional)* | **string** | If an attribute of 'fixed' is on the element a fixed grid will be uses instead |
