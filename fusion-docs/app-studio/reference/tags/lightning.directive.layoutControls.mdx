## Description

The `<layout:controls>` tag wraps control elements into a group making
all the controls on one line or individually for a control to be on a
separate line.

Layout controls is required for the following directives:-

* [Sort Tag](/app-studio/4.2/1122/search-sort)
* [Results Per Page
  Tag](/app-studio/4.2/1091/search-results-per-page)
* [Fields Tag](/app-studio/4.2/1116/search-fields)

## Usage

as element:

```xml
<layout:controls>
</layout:controls>
```
