## Description

Creates an animation when the value (on) is available.

## Usage

as element:

```xml
<layout:animate
       animation="{string}"
       on="{object}">
</layout:animate>
```

as attribute

```xml
<ANY layout-animate
     animation="{string}"
     on="{object}">
   ...
</ANY>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| animation | **string** | The animation to use when the value is available |
| on | **object** | The object to watch for a value |

```js

            $scope.fadeInUp = undefined;

            } else {
```
