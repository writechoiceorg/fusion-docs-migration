## Description

Redirects the user to `<app context>/logout/` when clicked

## Usage

as element:

```xml
<widget:logout>
</widget:logout>
```

### Directive info

* This directive creates new scope.

## Example

#### Source

```xml
<widget:logout></widget:logout>
```
