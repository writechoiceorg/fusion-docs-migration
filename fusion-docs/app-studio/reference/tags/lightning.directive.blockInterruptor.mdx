## Description

Creates an interrupter box with the transcluded content

## Usage

as element:

```xml
<block:interruptor>
</block:interruptor>
```

## Example

#### Source

```xml
<block:interruptor>
    <h2>Heads Up</h2>
    <p>I'm interrupting you!</p>
</block:interruptor>
```
