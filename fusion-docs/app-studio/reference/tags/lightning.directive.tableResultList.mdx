## Description

Creates a table

The following tags can be nested within the table result list tag.

* [Table Header](/app-studio/4.2/1067/table-header)
* [Table Header Cell](/app-studio/4.2/1128/table-header-cell)
* [Table Body](/app-studio/4.2/1176/table-body)
* [Table Row](/app-studio/4.2/1060/table-row)
* [Table Cell](/app-studio/4.2/1172/table-cell)

## Usage

as element:

```xml
<table:result-list
       response="{string}"
       results="{array}">
</table:result-list>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| response | **string** | The name of the response to populate the table with. (Use results or response not both.) |
| results | **array** | A list of results to be rendered in the table. (Use results or response not both.) |
