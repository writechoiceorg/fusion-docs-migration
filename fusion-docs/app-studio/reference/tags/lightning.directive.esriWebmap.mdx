## Description

Adds webmap layers from an esri webmap. The webmap id must be applied to
the geo-map element.

## Usage

as element:

```xml
<esri:webmap
       [url="{string}"]
       [toggle-layers="{boolean}"]>
</esri:webmap>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| url  *(optional)* | **string** | The URL to the arcgis portal, if not supplied the default will be used by esri. |
| toggle-layers  *(optional)* | **boolean** | Whether layers can be turned on and off. |

## Example

#### Source

```xml
<h3 id="example_esri-map">Esri Map</h3>
<geo:map response="response2" provider="esri" address-field="address" label-field="name" webmap-id="ef9c7fbda731474d98647bebb4b33c20" zoom-level="13">
    <esri:webmap toggle-layers="true"></esri:webmap>
</geo:map>
```
