## Description

Extends the HTML style attribute to apply twigkit classes to the element
as specified in the style attribute

## Usage

as attribute

```xml
<ANY styling="{string}">
   ...
</ANY>
```

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| styling | **string** | Space separated list of styles to apply to the element |
