## Description

The title directive allows you to set the title for the page.

## Usage

as element:

```xml
<helper:title
       title="{string}">
</helper:title>
```

### Directive info

* This directive creates new scope.

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| title | **string** | The title of the page. |

## Example

`<helper:title title="Twigkit Application" </search:title>`
