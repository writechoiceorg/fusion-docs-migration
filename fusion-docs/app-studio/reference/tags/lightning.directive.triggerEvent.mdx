## Description

Triggers $rootScope event on click

## Usage

as attribute

```xml
<ANY trigger-event="{string}"
     [trigger-delay="{string}"]>
   ...
</ANY>
```

### Directive info

* This directive creates new scope.

#### Parameters

| Param | Type | Details |
| --- | --- | --- |
| trigger-event | **string** | Name of the event to trigger |
| trigger-delay  *(optional)* | **string** | Milliseconds to delay event trigger |
