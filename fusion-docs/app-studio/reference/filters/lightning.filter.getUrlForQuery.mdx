## Description

Returns the query parameters for a query object

## Usage

### In HTML Template Binding

```
{{ getUrlForQuery_expression | getUrlForQuery }}
```

### In JavaScript

```js
$filter('getUrlForQuery')(query)
```

#### Parameters

| --- | --- | --- |
| query | Object | The query object |
