## Description

Renders fields actual value

## Usage

### In HTML Template Binding

```
{{ encodeURIComponent_expression | encodeURIComponent }}
```

### In JavaScript

```js
$filter('encodeURIComponent')(string)
```

#### Parameters

| --- | --- | --- |
| string | string | The value to encode |
