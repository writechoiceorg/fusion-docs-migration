## Description

Strip HTML from a string.

## Usage

### In HTML Template Binding

```
{{ htmlToPlainText_expression | htmlToPlainText }}
```

### In JavaScript

```js
$filter('htmlToPlainText')(string)
```

#### Parameters

| --- | --- | --- |
| string | string | The value to strip the html from. |
