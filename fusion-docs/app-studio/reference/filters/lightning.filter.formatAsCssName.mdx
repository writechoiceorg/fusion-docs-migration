## Description

Renders a string in a friendly name for CSS.

## Usage

### In HTML Template Binding

```
{{ formatAsCssName_expression | formatAsCssName }}
```

### In JavaScript

```js
$filter('formatAsCssName')(string)
```

#### Parameters

| --- | --- | --- |
| string | string | The string to format. |
