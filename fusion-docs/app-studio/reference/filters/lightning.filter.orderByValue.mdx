## Description

Orders values from facet or results

## Usage

### In HTML Template Binding

```
{{ orderByValue_expression | orderByValue }}
```

### In JavaScript

```js
$filter('orderByValue')([desc])
```

#### Parameters

| --- | --- | --- |
| desc *(optional)* | boolean | Whether it should be descending or not. |
