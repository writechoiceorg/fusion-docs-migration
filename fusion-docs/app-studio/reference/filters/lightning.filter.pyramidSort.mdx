## Description

Sorts values in a pyramid fashion

## Usage

### In HTML Template Binding

```
{{ pyramidSort_expression | pyramidSort:field }}
```

### In JavaScript

```js
$filter('pyramidSort')(items, field)
```

#### Parameters

| --- | --- | --- |
| items | object | Array of values |
| field | string | The field to sort on. |
