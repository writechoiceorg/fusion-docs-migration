**How does AppKit support responsive design?**  
All standard Appkit templates are responsively designed and will work correctly on tablets and mobiles with full feature parity.

**What is the process we must follow to accomplish responsive design?**  
As long as the existing templates are not compromised, responsive design will work out of the box. The user interface being adapted is a customization of a fully responsive template.
