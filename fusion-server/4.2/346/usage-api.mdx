The Usage API sends system usage data to Lucidworks. See [System Usage Monitor](/fusion-server/4.2/128/system-usage-monitor) for details.

<Tip>
**Important**

The Usage API is removed in Fusion 5.x.
</Tip>

## Examples

*Get the latest data that will be sent to Lucidworks on the next run:*

**INPUT**

```bash
curl -u USERNAME:PASSWORD https://FUSION_HOST:8764/api/usage
```
