Fusion API services are designed to be accessed via Fusionâ€™s authentication proxy module which is part of the Fusion UI service (default port 8764). All applications should use this method to access the API service:

```
https://FUSION_HOST:8764/api/<endpoint>
```

## List all Fusion component services

The Fusion `introspect` endpoint lists basic information about endpoints and parameters for all Fusion endpoints,
including the Connectors services endpoints:

```bash
curl -u USERNAME:PASSWORD https://FUSION_HOST:8764/api/introspect
```

## Assign Fusion request IDs
