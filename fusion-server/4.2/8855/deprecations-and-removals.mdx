## Active Deprecations

The expected removal versions listed in the **Active Deprecations** table below may not correspond to a planned release version. Removals take place *on or after* the expected removal version. If the expected removal is listed as 4.1.4, but Fusion 4.1.4 is not a planned release, the removal may take place in Fusion 4.2.0.

<Tip>
**Fusion Connectors deprecations and removals**

Deprecation and removal details for Fusion Connectors are found at [Fusion Connectors Deprecations and Removals](/fusion-connectors/ft39rr/deprecations-and-removals).
</Tip>

| Feature | Deprecated | Expected Removal | Notes |
| --- | --- | --- | --- |
| Webapps API | 4.2.0 | 4.3.0 | Webapps Appkit API |
| Experiments API | 4.0.0 | 4.1.0 | Experiments are now managed through the `/spark` endpoints. |
| Aggregator API | 4.0.0 | 4.1.0 | Aggregations are now managed through the `/spark` endpoints. |
