The Analytics Catalog query pipeline stage lets you define views of data stored in Fusion collections. You can query the objects in the Analytics Catalog using SQL, [Solr Streaming Expressions](https://cwiki.apache.org/confluence/display/solr/Streaming+Expressions), or regular Solr queries. See also the [Catalog API](/fusion-server/4.2/329/catalog-api).
